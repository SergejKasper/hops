render

"module:lib/browser.render"

(static) render(overrides) → {function}

hops' main function:

  • creates a Redux store
  • sets up React Router
  • handles rendering both in the browser and in node.

Using it is mandatory and its output must be the default export of your main module. And it's a little magic. In addition to routes and reducers, an html mountPoint selector and some othes may be passed as options.

Parameters:
